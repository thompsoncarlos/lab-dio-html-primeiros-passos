Estrutura Básica do HTML

Um documento HTML é dividido principalmente em duas partes:

  • HEAD: Contém as informações sobre o documento HTML.
    • Por exemplo, o título da página, versão do HTML, metadados, etc.
  • BODY: Contém tudo o que você deseja exibir na página da Web.

Falando sobre Tags

A anatomia básica de um elemento HTML consiste na tab de abertura cercada pelos sinais de menor < e pelo sinal de maior > e a tag de fechamento com sinal de / depois do sinal de menor. Entre as tags de abertura e fechamento em algumas tags pode ser inserido o conteúdo que refletirá o efeito da tag utilizada.

Atributos das Tags

Os atributos possibilitam dar um sentido maior ou classificar uma tag, não visível ao usuário na tela. Neste exemplo o atributo class permite identificar que esta tag terá um comportamente que se repetirá no site possibilitando acrescentar estilo com css e dinâmica com javascript.

Textos

Serão apresentadas tags básicas que ajudam na formatação de textos como exemplo:

Títulos (<h1>)

A formatação representando o nível de importância, com títulos e subtítulos <h1> até <h6>

                
                    <h1>Título 1 </h1>
                    <h2>Título 2 </h2>
                    <h3>Título 3 </h3>
                    <h4>Título 4</h4>
                    <h5>Título 5</h5>
                    <h6> Título 6</h6>
Resultado:

Título 1

Título 2

Título 3

Título 4

Título 5
Título 6

Parágrafos (<p>)

Adicionando um espaço acima e abaixo do texto (padding).

Blockquote (<blockquote>)

Permite inserir uma citação de outra fonte (<cite>), cujo endereço não fica visível ao usuário e o texto fica com um espaçamento interno (padding).

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

Listas Ordenadas e Não Ordenadas

A Lista Ordenada é representada pela tag <ol> que significa Ordered List, apresenta uma lista com marcação numérica da ordem em que são apresentadas, e cada um dos itens desta lista precisam estar entre a tag <li> que significa List Item.
Como exemplo dos passos de uma receita de bolo em que há uma ordem a ser executados. 

                
                <ol> 
                    <li>Misture os ingrediente no liquidificador</li>
                    <li>Unte a forma com manteiga e um pouco de farinha de trigo</li>
                    <li>Acrescente a mistura na forma</li>
                    <li>Leve ao forno durante 25min com 220o graus</li>
                </ol>
Resultado:
  1. Misture os ingrediente no liquidificador
  2. Unte a forma com manteiga e um pouco de farinha de trigo
  3. Acrescente a mistura na forma
  4. Leve ao forno durante 25min com 220o graus

A Lista Não Ordenada é representada pela tag <ul> que significa Unordered List, apresenta uma lista apenas com marcação onde a ordem dos itens não é necessária ser representada, e cada um dos itens desta lista precisam estar entre a tag <li> que significa List Item.
Como exemplo dos ingrediente de uma receita de bolo. 

                
                <ul> 
                    <li>Coloque 250g de farinha</li>
                    <li>Acrescente 3 ovos</li>
                    <li>Bata no liquidificador</li>
                </ul>
Resultado:
  • 250ml de leite
  • 250g de Farinha de Trigo
  • 3 ovos
