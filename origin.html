Da necessidade de realizar cálculos matemáticos desde antes de cristo até os dias atuais, foram criadas ferramentas e equipamentos para ajudar a aumentar a velocidade de resposta destes cálculos e possibilitar o emprego destes resultados em diversas áreas de atuação do ser humando. Começando pelo Ábaco e evoluindo cada vez mais com equipamentos que ocupavam salas inteiras movido a válvula depois

Comunicação Clients x Servers?

Os Clients são as pessoas que consomem a informação da internet por meio de algum dispositivo no navegador(browser), essa solicitação (requisição) de informações são encaminhadas aos servidores que cuidam de buscar responder a estas requisições e diferentes fontes de bados, base de dados, outros servidores.

Front-end x Back-end

Front-end (client-side): São todas as tecnologias que atuam em linha de interface de usuário para promover menus, formulários, blocos de conteúdo, barras de navegação, campos de busca, entre outros elementos de comunicação e GUI (Graphical User Interface). No contexto web as linguagens front-end prescrevem a comunicação entre browser e usuário. Isso significa que tudo o que é possível ver e interagir ao navegar entre as páginas de um website foi construído por uma ou mais ferramentas front-end. Nesta página, por exemplo, podemos acessar alguns links, visualizar algumas imagens e movimentar o conteúdo para cima e para baixo. Isso é tecnologia front-end. Atualmente a tríade mais consagrada entre os programadores é HTML, CSS e JavaScript – três linguagens que um desenvolver front-end precisa conhecer muito bem. Existe ainda bibliotecas e frameworks como Bootstrap, jQuery, Knockout, Backbone.js e D3.js que auxiliam na construção desses projetos.

Back-end (server-side): São todas as tecnologias inseridas em processamento de dados e interação com as bases de negócios de um sistema. Em outras palavras, trata-se da funcionalidade interna de um programa. Por essa razão tudo o que estiver regrado a segurança, performance, upgrade e manipulação de dados faz parte da camada back-end. Na programação web o back-end estabelece uma transição de dados entre browser e servidor que não pode ser compreendida pelo usuário final. Antes de acessar essa página, por exemplo, uma busca pelo IP no gTLDs e ccTLDs do servidor DNS (Domain Name System) foi realizado e, após encontrado, fez com que o servidor enviasse essas informações que foram interpretadas pelo browser e renderizadas nesse conteúdo. Essa sequência de eventos aconteceu de modo oculto, ou seja, de modo imperceptível à interface gráfica do utilizador. Isto é tecnologia back-end. Atualmente as principais linguagens back-end são Java, PHP, C#, Go, ASP, Swift e Python. Todas elas contam com várias bibliotecas e frameworks como Laravel, Eden, Apache Wicket, NHibernate, Django e Gin que facilitam o trabalho do programador.
